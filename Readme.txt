This project implements the Little Lemon restaurant API.  
Below are the endpoints available for peer testing. 

1. Menu API (no authentication required)  
   - GET /api/menu-items/  
   - POST /api/menu-items/  
   - GET /api/menu-items/<id>/  
   - PUT/PATCH /api/menu-items/<id>/  
   - DELETE /api/menu-items/<id>/  

2. Booking API (requires authentication token)  
   - GET /restaurant/booking/tables/  
   - POST /restaurant/booking/tables/  
   - GET /restaurant/booking/tables/<id>/  
   - PUT/PATCH /restaurant/booking/tables/<id>/  
   - DELETE /restaurant/booking/tables/<id>/
